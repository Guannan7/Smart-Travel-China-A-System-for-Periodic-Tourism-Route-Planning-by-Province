# 智游中国：省域周期性旅游路线规划系统

## 项目介绍
智游中国是一个智能旅游路线规划系统，专注于城市级深度旅游规划。系统能够根据用户输入的省份/城市、旅游天数和偏好标签，为用户生成量身定制的旅游行程。

## 核心功能
1. **用户输入与识别模块**：支持省份/城市自动识别，精确控制城市粒度
2. **景点数据库模块**：城市级景点库，包含详细信息和城市筛选功能
3. **智能路线规划引擎**：基于地理邻近性、主题一致性和时间约束的规划算法
4. **周期性优化模块**：季节适配和节假日适配
5. **行程详情输出模块**：详细的每日行程安排和本地建议
6. **可视化与导出模块**：地图展示和行程导出功能

## 特色亮点
- **城市级精准规划**：用户输入具体城市时，仅规划该城市景点
- **市内交通建议**：提供地铁、公交、共享单车等市内交通方案
- **聚焦单一区域**：住宿推荐聚焦城市核心区域
- **适合深度游的行程密度**：每天安排2-4个景点，提供更深入的城市体验

## 技术栈
- 后端框架：Flask
- API集成：deepseek、阿里云、高德地图
- 数据处理：Pandas、NumPy
- 地理位置处理：GeoPy
- 地图可视化：Folium

## 安装说明
1. 克隆项目仓库
2. 安装依赖：`pip install -r requirements.txt`
3. 配置API密钥（详见.env.example）
4. 运行应用：`python app.py`

## 使用示例
用户输入：云南省，大理市，3天
系统输出：大理市深度游3天行程，包含大理古城、洱海、双廊古镇等景点的详细安排。

## 注意事项
- 系统需要有效的API密钥才能正常运行
- 行程规划基于实时数据，建议出行前再次确认景点开放信息
- 餐饮和住宿推荐仅供参考，建议提前预订
